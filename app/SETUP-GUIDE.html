Launch Setup Guide

Step-by-step instructions to go from these files to a live website. Estimated time to launch: 2–4 hours.

What You Built

The app/ folder contains a complete Next.js web application for Steadfast Student Services. Here is what is included:

  • Landing page — Hero, stats, services, process, testimonials, CTAs
  • Services page — All three risk tiers with feature lists and FAQ
  • Free Risk Assessment Quiz — 7-step questionnaire with AI scoring
  • Booking page — Cal.com embed for free consultations
  • Student Portal — Progress tracker, document checklist, application Kanban
  • AI Chatbot — "Sofia" powered by Gemini 1.5 Flash (free tier)
  • Contact page — Form with automatic lead creation and welcome email
  • About page — Story, stats, values
  • Resources/Blog — Article hub with downloadable checklists
  • Admin Dashboard — CRM lead pipeline with search

Step-by-Step Launch Checklist

STEP 1 — Install Node.js

Download Node.js (if not already installed)

Go to nodejs.org and download the LTS version (v20 or higher). Verify it works:

node --version npm --version
STEP 2 — Install Dependencies

Open terminal in the app folder

In VS Code: Terminal → New Terminal. Or use Windows Terminal / PowerShell.

cd "c:\Users\DueDiligence\Desktop\student enrollment\app" npm install

This downloads all packages (Next.js, Supabase, Tailwind, Gemini SDK, etc.). Takes 2–3 minutes.

STEP 3 — Create Supabase Project FREE

Set up your database and auth

  1. Go to supabase.com → Sign up → New Project
  2. Name it "steadfast-student-services", choose a region (US East recommended)
  3. Go to Project Settings → API → copy Project URL and anon public key
  4. Go to SQL Editor → paste contents of supabase-setup.sql → Run
  5. Go to Storage → New Bucket → name: student-documents → Private
STEP 4 — Get Gemini API Key FREE

Free AI chatbot — 1 million tokens per day

  1. Go to aistudio.google.com
  2. Sign in with Google → click "Get API Key" → Create API Key
  3. Copy the key (starts with AIzaSy...)
STEP 5 — Set Up Resend Email FREE

3,000 free emails per month

  1. Go to resend.com → Sign up
  2. Add your domain (or use the Resend test domain to start)
  3. API Keys → Create API Key → Copy it
STEP 6 — Create Environment File

Copy .env.example to .env.local and fill in your values

copy .env.example .env.local

Then open .env.local and fill in:

  • NEXT_PUBLIC_SUPABASE_URL — from Supabase dashboard
  • NEXT_PUBLIC_SUPABASE_ANON_KEY — from Supabase dashboard
  • SUPABASE_SERVICE_ROLE_KEY — from Supabase Settings → API
  • GEMINI_API_KEY — from Google AI Studio
  • RESEND_API_KEY — from Resend dashboard
  • RESEND_FROM_EMAIL — your verified sender email
STEP 7 — Test Locally

Run the development server

npm run dev

Open http://localhost:3000 in your browser. You should see the landing page. Test:

  • Landing page loads with navigation and hero
  • AI chatbot button appears bottom-right — click it, send a message
  • Assessment quiz completes all 7 steps
  • Contact form submits without errors
  • Services, About, Resources pages load
STEP 8 — Deploy to Vercel FREE

Go live in 5 minutes

  1. Create a GitHub account if you don't have one (github.com)
  2. Push your app/ folder to a new GitHub repository
  3. Go to vercel.com → Sign up with GitHub → Import your repo
  4. Vercel auto-detects Next.js — no config needed
  5. Add all your environment variables in Vercel dashboard → Settings → Environment Variables
  6. Click Deploy — your site is live in 2 minutes!
STEP 9 — Connect Your Domain FREE via Cloudflare

Optional: Add a custom domain

  1. Buy steadfaststudentservices.com from Namecheap (~$12/year)
  2. Sign up at cloudflare.com → Add Site → follow DNS setup
  3. In Vercel: Settings → Domains → Add your domain
  4. Cloudflare provides free CDN + SSL + DDoS protection
STEP 10 — Set Up Cal.com Booking FREE

Enable online consultation booking

  1. Go to cal.com → Sign up free
  2. Create event type: "Free 30-Minute Consultation" (30 min, video or phone)
  3. Connect your Google Calendar or Outlook
  4. Get your embed URL → update the src in /app/book/page.tsx
  5. Set up webhook: Cal.com → Settings → Webhooks → point to yoursite.com/api/notify
